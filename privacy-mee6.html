Privacy Policy for N G C [MEE6 Premium]

Effective Date: November 24th, 2025
Last Updated: November 24th, 2025

Definitions and Preface

In this document, the "Bot" refers to the custom Discord application and its associated services provided through the "Custom Bot" feature, which is a premium, lifetime subscription benefit of the MEE6 service provided by Sidescroll Ventures SAS. "You" refers to the user interacting with the Bot within the Discord server. "We," "our," and "the server" refer to the administration of the Nexus Gaming & Café Discord server.

The Bot is a dedicated tool for a single Discord server (Nexus Gaming & Café) and is not a public service. By interacting with the Bot and the server, you consent to the data practices outlined in this policy. Your use is also subject to the MEE6 Privacy Policy and MEE6 Terms of Use.

The Bot's operator reserves the right to modify this privacy policy at any time. Significant changes will be announced in the Nexus Gaming & Café Discord server.

MEE6 Premium Integration

This Bot operates as a custom instance through MEE6's "Custom Bot" feature under a lifetime premium subscription. As such:

  • All data processing occurs through MEE6's secure infrastructure
  • Your data is subject to MEE6's data protection measures and policies
  • Premium features are delivered through MEE6's platform and servers
  • We utilize MEE6's enhanced capabilities for bot functionality

Collected Data and Rationale

The Bot collects and processes the following types of data through the MEE6 platform:

Discord User and Server Data: The Bot receives information provided by Discord's API including:

  • User IDs, usernames, discriminators, and avatar data
  • Server member lists, roles, and channel information
  • Message content and timestamps from monitored channels
  • Voice channel participation data

Premium Feature Data: Through our MEE6 lifetime subscription, the Bot processes data for enhanced features including:

  • Extended custom commands and user interactions
  • Reaction roles and user preference data
  • Economy system transactions and balances
  • Social media alert preferences
  • Moderation actions and audit logs
  • Ticketing system conversations
  • Level progression and XP data

Temporary Data Caching:

  • Command responses and interaction data
  • Member count and server statistics
  • Temporary voice data for premium features

Data Storage and Security

MEE6 Infrastructure: All data is processed and stored through MEE6's secure infrastructure in compliance with their data protection standards.

Local Storage:

  • Temporary cache data with automatic expiration
  • Configuration settings and preferences
  • Server-specific customization data

Security Measures:

  • Data transmission encrypted through Discord and MEE6
  • Access restricted to authorized server staff
  • Regular security updates through MEE6 platform

Data Retention Periods

  • Temporary Cache: Automatically cleared every 5-30 minutes
  • User Data: Removed upon user leaving the server
  • Moderation Logs: Retained for 90 days
  • Premium Feature Data: Subject to MEE6's retention policies
  • Voice Data: Automatically deleted within 24 hours

Your Rights

You have the right to:

  • Know what data is stored about you
  • Request access to your personal data
  • Request correction of inaccurate data
  • Request deletion of your personal data
  • Object to processing of your data

To exercise these rights, contact server administration through the Nexus Gaming & Café Discord server. For MEE6-specific data concerns, contact hi@mee6bot.com.

Data Sharing

The Bot shares data in these contexts:

  • MEE6 Platform: Essential data shared for premium functionality
  • Discord: Subject to Discord's Privacy Policy
  • Server Administration: Authorized staff for operational purposes

No data is sold, rented, or traded with third parties for marketing purposes.
