Definitions and Preface

In this document, the "Bot" refers to the Discord application and its associated code designed to provide moderation, utility, and management services for the "Nexus Gaming & Caf√©" Discord server. "You" refers to the user interacting with the Bot within the Discord server. "We," "our," and "the server" refer to the administration of the Nexus Gaming & Caf√© Discord server.

The Bot is a dedicated tool for a single Discord server (Nexus Gaming & Caf√©) and is not a public service. By interacting with the Bot and the server, you consent to the data practices outlined in this policy.

The Bot's operator reserves the right to modify this privacy policy at any time. Significant changes will be announced in the Nexus Gaming & Caf√© Discord server.

Collected Data and Rationale

The Bot collects and processes the following types of data to function:

  1. Discord User and Message Data: The Bot inherently receives information provided by Discord's API when you use its commands or interact in channels it monitors. This includes, but is not limited to:
    • User IDs, usernames, and discriminators.
    • Server member lists and roles.
    • Message content, channel IDs, and message timestamps from channels where the Bot's features are active (e.g., snipe logging, message forwarding).
    • Information related to moderation actions (kicks, bans, timeouts, warnings) including the user ID of the staff member, the target user, the reason, and the timestamp.
  2. Moderation & Logging: To maintain server security and a record of actions, the Bot logs moderation events (bans, kicks, warns, timeouts, nickname changes) to a designated moderator channel. These logs include user IDs and the reasons for the action.
  3. Message Forwarding: The Bot forwards messages from specific source channels to specific destination channels, both within the same server and to other servers via webhooks. The content of these messages, including text and attachments, is processed for this purpose.
  4. Temporary Data Caching:
    • Snipe Functionality: Deleted message content, author IDs, and timestamps from a specific channel are temporarily stored in the Bot's active memory for a short period (5 minutes) to allow moderators to review recently deleted messages via the /snipe command. This data is not permanently stored on disk.
    • Member Count: The server's member count is cached for 5 minutes to reduce API load when updating a stats channel.
  5. Direct Messages (DM): When a moderator uses the /dm command, the specified message content is sent to a user. A log of this action, including the sender, recipient, and message content, is sent to a designated webhook for administrative oversight.

The Bot does not intentionally collect sensitive personal information such as real names, addresses, or financial data. The data collected is strictly limited to what is necessary for the Bot's features to operate within the Discord platform.

Data Storage and Security

  • Volatile Data: Data for the snipe command is stored in the Bot's temporary memory (RAM) and is not written to a permanent file. It is automatically cleared after its expiration time.
  • Persistent Logs: Moderation actions and other operational logs may be written to a file (bot.log) and sent to a Discord webhook for permanent record-keeping by the server administration.
  • Environment Variables: Sensitive data like API tokens and channel IDs are stored as environment variables and are not hardcoded.
  • Access: Access to logs and data is restricted to server administrators and moderators with the appropriate permissions.

  • Request the deletion of your data.

To exercise these rights, please contact the server administration directly through the Nexus Gaming & Caf√© Discord server. Please note that we may need to verify your identity as the Discord user in question. Data Sharing

The Bot shares data in the following specific, functional contexts:

  • Message Forwarding: Message content is shared with other channels/servers as configured by the server administration.
  • Webhook Logging: Logs of actions (moderation, DMs) are sent to Discord webhooks, which are controlled by the server administration.
  • Discord: As a Discord bot, it is subject to Discord's Privacy Policy for any data passed through its platform.

The Bot does not sell, rent, or trade your data with any third parties for marketing or commercial purposes.
